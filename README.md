# ENDOAI
**ENDOAI is an advanced AI toolkit for the diagnosis and surgical management of endometriosis.**  
We leverage cutting-edge deep learning techniques to assist clinicians and surgeons in:
- Detecting and segmenting peritoneal endometriosis, deep infiltrating endometriosis (DIE), and ovarian endometriomas.
- Mapping risk zones for surgical planning.
- Providing 2D/3D visualizations for preoperative decision-making.

## 🌟 Key Features
- **AI-Powered Diagnosis**: Automated lesion detection and segmentation from MRI/ultrasound.
- **Surgical Guidance**: Risk mapping for critical structures like nerves and ureters.
- **Custom Visualizations**: Advanced 3D reconstructions for surgical planning.
